# instabuyModules
Instabuy modules for all projects inside instabuy.

## Installation 
```sh
npm install instabuymodules --save
yarn add instabuymodules
bower install instabuymodules --save
```
## Usage
### Build 
```sh
npm run build
```
### Test 
```sh
npm run test
```
### Coverage 
```sh
npm run cover
```
